# 设计模式
23种设计模式深入学习 1.创建型 2.结构型 3.行为型

### 1.创建型
对类的现实化进行了抽象，能够使软件模块做到与对象的创建和组织无关。

---

### 2.结构型
描述类和对象之间如何进行有效的组织，以形成良好的软件体系结构，主要的方式是使用继承关系来组织各个类。

---

### 3.行为型
类和对象之间如何交互以及如何分配职责
